 ## Import Instructions for Figma/FigJam
 
 ### Option 1: Direct Copy
 1. Copy the ASCII diagram above
 2. Open FigJam → Create new board
 3. Paste directly - FigJam will render it as a code block
 4. Use as reference to recreate with FigJam shapes
 
 ### Option 2: Use as Wireframe Reference
 1. Create a new FigJam file
 2. Use sticky notes for each screen
 3. Use connectors to show flow
 4. Color-code: Blue = Free, Purple = Pro features
 
 ### Suggested FigJam Colors
 - **Primary Screens**: `#7C3AED` (Violet)
 - **Actions/CTAs**: `#EC4899` (Magenta)
 - **Free Features**: `#3B82F6` (Blue)
 - **Pro Features**: `#F59E0B` (Amber with lock icon)
 - **Mobile Elements**: `#10B981` (Emerald)

 ## Quick Reference: Route Map
 
 | Route | Screen | Auth Required |
 |-------|--------|---------------|
 | `/` | Landing Page | No |
 | `/auth` | Login/Signup | No |
 | `/dashboard` | Project List | Yes |
 | `/dashboard/:id` | Output View | Yes |
 | `/settings` | User Settings | Yes |
 | `/pricing` | Pricing Page | No |
 | `/p/:id` | Public Presentation | No |
 | `/practice` | Teleprompter Mode | Yes (Pro) |
